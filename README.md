# Agro Forte, Futuro Sustentável — Agrinho Paraná 🌾🚜

Este é um projeto desenvolvido para o **Programa Agrinho Paraná (Subcategoria 3)**. A aplicação consiste em uma página web interativa dedicada a conscientizar e demonstrar o equilíbrio ideal entre a alta produtividade no campo e a preservação do meio ambiente no estado do Paraná.

---

## 📌 Sobre o Projeto

O **Agro Sustentável** destaca a importância da inovação tecnológica no campo como a principal aliada da sustentabilidade. Através de recursos visuais e interativos, a página aborda como o uso de tecnologias modernas (como drones e sensores de solo) ajuda a evitar desperdícios, proteger a biodiversidade e garantir recursos para as próximas gerações.

## 🚀 Funcionalidades

- **Seções Informativas:** Explicações claras sobre os três pilares fundamentais: Produção Forte, Meio Ambiente e Sustentabilidade.
- **Interatividade de Curiosidades:** Um botão dinâmico que exibe fatos e curiosidades sobre a tecnologia aplicada ao agronegócio.
- **Simulador de Práticas Sustentáveis:** Uma ferramenta interativa onde o usuário seleciona uma ação agrícola (como Plantio Direto, Rotação de Culturas ou Uso de Drones) e descobre o impacto positivo gerado na fazenda.
- **Design Responsivo:** Interface otimizada para visualização em computadores, tablets e smartphones.

## 🛠️ Tecnologias Utilizadas

As seguintes tecnologias e bibliotecas foram utilizadas na construção deste front-end:

* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) — Estruturação do conteúdo da página.
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) — Estilização, layout e responsividade (arquivo `style.css`).
* [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — Comportamento interativo do simulador e botão de curiosidades (arquivo `script.js`).
* [Font Awesome](https://fontawesome.com/) — Biblioteca de ícones modernos e dinâmicos para a interface.
